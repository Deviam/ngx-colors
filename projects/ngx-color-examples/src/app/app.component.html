Overview

ngx-colors is an angular components that allows users to select a color either through text input(hex, rgba, hsla), by choosing a pre set color from the palette, or from Hue, Lightness, Alpha sliders.

Install & Import

Npm

Install dependency
npm install @angular/animations

Install ngx-colors
npm install ngx-colors

Import

import {{'{'}} NgxColorsModule {{'}'}} from 'ngx-colors';
  
        @NgModule({{'{'}}
          ...
          imports: [
            ...
            NgxColorsModule
          ]
        {{'}'}})

Basic usage

Getting and setting a colors

The <ngx-colors> element supports 2-way binding to the color property.
<ngx-colors [(color)]="ioColor"></ngx-colors>

Detecting changes

Use the property change to detect changes when the user interact with the color picker
<ngx-colors (change)="updateGradient()" [(color)]="ioColor"></ngx-colors>
