Overview

ngx-colors is an angular component that allows users to select a color either through text input(hex, rgba, hsla), by choosing a pre set color from the palette, or a color picker with using Hue, Lightness and Alpha sliders.

It is composed of two parts:
  • ngx-colors-trigger: This directive can be applied to any html element turning it into a trigger that when clicked will open the color picker
  • ngx-colors: This component is a premade button that will display the selected color.
